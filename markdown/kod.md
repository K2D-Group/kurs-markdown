# Kod

Jeśli chcemy zapisać fragment kodu w oryginalnej formie kodu, 
należy umieścić go miedzy znakami lewych apostrofów (\`).

## Przykład
```
Użyj funkcji `printf()`
```
wynik:

Użyj funkcji `printf()`
